What It Does

Auto-Save
Every member returned on Refresh is saved automatically. No manual saving needed ‚Äî the list builds itself.
Class Color
Names colored by class using standard WoW colors, learned from your guild roster and friends list.
Zone
Grey zone text under each name, refreshed on every window open. Requires guild or friends list membership.

Green dot ‚Äî currently in channel. Grey dot ‚Äî saved but not currently online in channel.

Interaction

Navigation

Selecting a Player

Click any row with Left Click to select that player. The row glows and the name appears in gold above the action buttons as Selected: Playername. The four action buttons activate based on the player's relationship with you.

Right-Click Context Menu

Playername
Whisper
Invite
Target
Cancel

Right-clicking any row opens a native WoW dropdown menu at your cursor with quick actions. No player selection required ‚Äî useful for fast whispers or targets without changing your current selection.

Action Button Grid

After selecting a player, use the 2√ó2 button grid. Buttons are greyed out when no player is selected, and Add Friend / Remove Friend are mutually exclusive ‚Äî only the relevant one is active.

Add Friend
Adds to WoW friends list
Send Message
Opens whisper in chat
Remove Friend
Removes from friends list
Group Invite
Invites to your group
Clear All Saved

Clear All Saved removes all players from your persistent list (confirmation required). Does not affect your WoW friends list.

Header Buttons

Join
Joins KnightGuard if not already in it. Automatically sets chat color to light blue and refreshes the member list once the channel registers.
Refresh
Re-queries the KnightGuard member list from the server. Any new members are automatically saved. The window also auto-refreshes when opened.

Opening the Window

KG Button
Draggable floating badge, action-bar sized. Click to toggle. Position is saved between sessions.
Minimap Icon
Gold KG icon on the minimap border ring. Click to open or close.
/cf
Slash command toggle ‚Äî works from anywhere in the game.
